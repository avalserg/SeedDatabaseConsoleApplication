## SeedDatabaseConsoleApplication

###  Консольное приложения для добавления через API 100 сгенерированных сущностей Patient.

#### Особенности

- работает с приложением Patient_Api_Test
- чтобы сгенерировать сущности необходимо запустить приложение при запущенном API 
- endpoint (https://localhost:5001/api/v1/Patient)
